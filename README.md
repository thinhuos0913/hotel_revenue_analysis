# Hotel revenue analysis
## Description
Develop hotel booking database using SQL Server Management Studio to visualize and analyze hotel revenue.
## Practice Skills
- Build a database (import data from Excel file to SQL Server).
- Retrieve and analyze data with SQL Command.
- Connect database to Power BI.
- Visualize data and create dashboard in Power BI.
